# ListRotater

##### First clone the repo:

```sh
git clone https://github.com/benthehorn/ListRotater.git
```
##### cd into the directory you cloned the repo into : 
---
```sh
 cd ListRotater
 npm install
```

###### Run the code
```sh
  node rotate.js "[1,2,3,4,5,6]" 2
``` 

It should return [3,4,5,6,1,2]

##### to run the tests
---
```sh
 npm test
```
